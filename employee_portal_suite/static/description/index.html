Employee Portal Suite

A complete Employee & Material Request system with GPS Attendance, Portal Announcements, Multi-Level Approvals, Interactive Dashboards, and Purchase Integration

Employee Portal Suite extends Odoo with a secure self-service portal for employees and a unified approval center for managers, HR, finance teams, and executives. It is designed for companies that require structured internal workflows, full traceability, and clean user experience.

Employee Portal Dashboard

Employees access a clean and intuitive portal dashboard where they can submit requests, track approval status, view documents awaiting signature, check in/out using GPS, and read portal announcements — all without accessing the backend.

Employee Requests Workflow

Employees can submit a wide range of internal requests such as leave, salary advance, training, medical reimbursement, asset requests, letters, transfers, and more.

Each request follows a structured approval flow:

  • Employee → Direct Manager
  • HR Approval
  • Finance Approval
  • CEO Approval

Requests are automatically numbered, tracked, and fully logged with approval dates and comments.

Material Requests Management

Material Requests allow employees to request materials for projects or worksites with full approval control. Each request includes worksite, delivery date validation, materials list, and accounting information. Approved material requests are now also linked to vendor bills (Account Move) for full financial traceability.

Approval stages are fully configurable and follow this flow:

  • Purchase Representative
  • Store Manager (Project-based)
  • Project Manager
  • Projects Director
  • CEO Approval

GPS Attendance with Geofencing

Employees can check in and check out directly from the portal using their device's GPS location. The system validates attendance against each employee's configured work location radius, ensuring check-ins and check-outs occur only within the allowed geofenced area.

Key attendance features include:

  • GPS-based Check-In / Check-Out from the portal
  • Geofencing per work location with configurable radius
  • Out-of-bounds checkout flagging (checkout_outside_location)
  • Automatic checkout cron for employees who forgot to check out (auto_checked_out)
  • Attendance history and status visible on the employee portal

Portal Announcements

Administrators can publish time-bound announcements directly on the employee portal. Announcements support rich HTML content, color-coded urgency levels (Blue, Green, Yellow, Red), date ranges, and visibility filtering by user group — ensuring the right message reaches the right audience.

  • Rich HTML message editor
  • Start and end date scheduling
  • Group-based visibility control
  • Color-coded priority (Info / Success / Warning / Danger)
  • Ordering by sequence for editorial control

Interactive Backend Dashboards

Both Employee Requests and Material Requests now include interactive Odoo backend dashboards built with OWL components. Dashboards display real-time KPIs, request statuses, and key metrics for managers and administrators at a glance.

  • Employee Requests Dashboard — status breakdown, pending approvals, recent activity
  • Material Requests Dashboard — open requests, approved/rejected counts, spending overview
  • Fully integrated into Odoo's backend assets pipeline (JS + XML + SCSS)

Approval Timeline & Transparency

Every request includes a detailed approval timeline visible in both the backend and the portal. Users can see exactly who approved each stage, when it was approved, and any comments added.

Rejections clearly show the rejection stage and reason, ensuring transparency and reducing follow-ups.

Purchase Order & Vendor Bill Integration

Once a Material Request is fully approved, the Purchase team can create Purchase Orders directly from the request. The system automatically links Purchase Orders to the originating Material Request, displays PO numbers, and shows the current PO status (RFQ, Sent, Approved, Received, etc.).

New in this version: approved Material Requests can also be linked to vendor bills (Account Moves), providing full financial traceability from request to payment.

Manager & Approvals Portal

Managers and approvers get a dedicated portal section where they can review pending Employee and Material Requests assigned to them.

Approvals are permission-based and respect project-specific roles such as Store Manager and Project Manager.

PDF Reports & Documentation

Each request type generates a professional PDF document including request details, materials, approval history, and signatures.

PDFs can be printed or shared directly from Odoo, ensuring proper documentation and audit readiness.

Roles, Security & Access Control

Employee Portal Suite uses dedicated security groups to ensure data isolation and correct access:

  • Employee Portal User
  • Manager
  • HR
  • Finance
  • Projects Director
  • CEO

Employees only see their own requests, while approvers see only items assigned to their role. Attendance records are similarly isolated per employee.
